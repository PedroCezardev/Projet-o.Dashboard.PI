Sobre Nós

Melhores análises e insights

Nossa equipe desenvolveu o projeto Dash Metrics para ser o melhor sistema de análise de dados escolares do Brasil, nele é possível ter o contato com todos os dados das escolas brasileiras, por exemplo: Números de alunos de qualquer região, quais as melhores médias de notas de escolas, ou simplesmente saber quais as escolas tem a maior quantidade de indígenas ou alunos negros.

Serviços

Moderno e versátil

O Dash Metrics é uma inovação muito importante para a educação no Brasil, ele vai oferecer todo os tipos de dados em forma de gráfico para facilitar o entedimento do analista e através disso será possível realizar diversas melhorias em várias escolas da região mais carente ou até em todas as regiões, possibilitando assim uma aplicação mais eficiente do dinheiro público.

Diferente e intuitivo

Além de ser bastante gamificado a nossa plataforma possui algumas características de acessibilidade, por exemplo: O Dash Metrics conta com várias funcionalidades para o usuário possa compreender qual análise aquele gráfico pode ser utilizado, também é possível alterar as cores para o modo "Dark" ou "Ligth" personalizando como preferir.

Inclusivo

Nossa aplicação é acessível a todos, por exemplo: Analista de dados (pessoas que trabalham com dashboard profissional), diretores acadêmicos, professores universitários, alunos interessados em realizar análises ou até gestores escolares.

Perguntas

+

É possível visualizar o desempenho acadêmico de alunos dentro da plataforma?

Sim a plataforma Dash Metrics possui diversos gráficos para visualizar as maiores e menores médias das escolas do Brasil.

+

É possível ver quais as matérias que tem as maiores notas?

Sim, é possível visualizar o nome das matérias que são ofertadas em cada escola e quais as notas dos alunos em cada matéria.

+

É possível ver as taxas de aprovação e reprovação?

Sim no Dash Metrics é possível ver a quantidade de aluno que foram aprovados ou não, em qualquer escola ou região.

+

Os gráficos gerados no Dashboard são interativos?

Nosso Dashboard possui gráficos bastante interativos, por exemplo, é possível filtrar no próprio gráfico quais dados você deseja analisar. Ou apenas passando o mouse em cima do gráfico, é possível analisar qual o dado e qual é a estatística do gráfico.

+

É possível Exportar ou Baixar os gráficos gerados?

A nossa plataforma disponibiliza, na barra de menu, a funcionalidade "Exportar Gráfico" que em apenas alguns cliques facilmente é possível baixar os gráficos gerados. Outra forma de fazer download é clicando com botão direito do mouse no gráfico que você deseja baixar e clicar em "Salvar imagem como...", dessa forma você consegue baixar os gráficos individualmente.
